# Portfolio (Frontend Template)

This is a modern, responsive, dark-themed portfolio template built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## Features
- Responsive layout (desktop/tablet/mobile)
- Dark neon teal accent color palette
- Sticky navbar with smooth scrolling
- Hero with profile graphic and typing effect
- About, Services, Portfolio, Skills, Certificates, Resume (PDF), Blog
- Simple JS form validation
- AOS scroll animations
- Google Fonts (Noto Serif)

## How to use
1. Open `index.html` in your browser or serve the folder with a static server (e.g., `npx http-server`).
2. Replace images in `assets/images/` with your own (keep the same filenames or update the HTML).
3. Update the name, copy, and social links in `index.html`.
4. Replace `assets/resume.pdf` with your real PDF (same filename) to enable downloads.
5. Test responsiveness by resizing the browser or using devtools (Desktop / Tablet / Mobile views).

## Files
- `index.html` - main markup
- `css/styles.css` - custom styles
- `js/main.js` - interactivity
- `assets/images/` - placeholder images
- `assets/resume.pdf` - placeholder resume (replace with your PDF)

## Notes
- No backend is included; contact form is client-side only (demo mode).
- Replace placeholder assets with real images for production.

Enjoy customizing your portfolio!
